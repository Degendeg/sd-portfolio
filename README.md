# My Portfolio

Welcome to my portfolio! This website showcases my work and provides information about me. It's built using React and hosted on Netlify.

## Features

- **About Me:** Learn more about my background, skills, and experiences.
- **Projects:** Explore the projects I've worked on and view their details.
- **Contact:** Reach out to me using the contact form.

## Technologies Used

- **Framework:** React
- **Build Tool:** Vite
- **Hosting:** Netlify
- **Contact Form:** Formspree

## Getting Started

To view the live website, visit [My Portfolio](https://your-portfolio-url.netlify.app/).

To run the project locally:

1. Clone this repository: `git clone https://github.com/your-username/sd-portfolio.git`
2. Navigate to the project directory: `cd sd-portfolio`
3. Install dependencies: `npm i`
4. Start the development server: `npm run dev`
5. Open your browser and visit `http://localhost:{port}` usually `5173`
